      this was created and tested on a Windows 11 machine, 
      so the following steps will also assume a Windows 11 machine


1. Download and install Node.js from: https://nodejs.org/en/download
2. Clone this repository
3. Open a command prompt to the root directory
4. run "npm install" (you may be prompted to install a newer version of npm)
5. run "npm run test"
6. The test will run and at the end will show a command to display the test report: "npx playwright show-report"
7. The test report will show each test result, which can be clicked to show step details and attachments
